
 Scripts SQL MySQL
Este repositorio contiene consultas SQL desarrolladas en MySQL orientadas a la extracción y análisis de datos para responder preguntas de negocio concretas y apoyar la toma de decisiones
  - **Joins**: Combinación y cruce de múltiples tablas para consolidar información de distintas fuentes.
  - **Limpieza de datos**: Preparación, normalización y tratamiento de datos como paso previo al análisis.
  - **Consultas avanzadas**: Agregaciones, filtros complejos, CTEs y funciones de ventana aplicadas a casos de análisis reales.
Puedes explorar los scripts en la carpeta [`scripts_sql`](https://github.com/estelatrdz/estelatr/tree/main/Scripts%20MySQL).
